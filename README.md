＃WordPress的OAuth 2.0

添加中文

将应用程序连接到您的WordPress网站，而无需提供密码。

该插件使用OAuth 2协议允许授权授权； 也就是说，允许应用程序使用一组二级凭据访问站点。 这允许服务器管理员控制哪些应用程序可以访问网站，并允许用户控制哪些应用程序可以访问其数据。

此插件仅支持WordPress> = 4.8。
